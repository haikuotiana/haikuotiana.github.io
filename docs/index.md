### 安装 & 使用

1. 通过 npm /yarn

```bash
# yarn add fast-utiljs
npm install fast-utiljs
```

2. 通过 cdn 引入

```html
<script src="https://cdn.jsdelivr.net/npm/fast-utiljs@1.1.1/dist/fast-utiljs.min.js"></script>
```
